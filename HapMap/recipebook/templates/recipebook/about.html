Wat is HapMap

HapMap.nl is een recepten website gemaakt door de Cookie van de BlauweSchuit. Alle recepten op de site zijn geschikt voor het gebruikt bij grote groepen. De meeste recepten zijn meerdere malen getest.

Help mee de HapMap nog beter te maken

Samen maken we we de hapmap natruurlijk nog veel beter. Op dit moment is het nog niet mogenlijk om zelf een account aan te maken. Maar wil je toch graag recepten toevoegen aan de hapmap kun je altijd een mailtje sturen aan mzondervan@hotmail.nl.

Versie historie

v0.0.2 stable

Nieuwe functies zijn leuk maar dan moeten ze wel een beetje stabiel zijn dus

  • Search funtie layout werkt nu niet meer met een dropdown
  • Recepten kunnen nu ook worden gefilterd op categorie
  • Aantal personen input werkt geeft nu een nete error melding als input leeg of geen getal is

v0.0.2 beta

Nu de site een soort van sabiel draait wordt het tijd voor leuke functies

  • Zoek functie op home page
  • Recepten kunnen nu een categrie hebben
  • De ingredient hoeveelheden kunnen nu berekent worden
  • Toevoegen van recepten is nu een stuk makelijker dankzij django admin

v0.0.1 stable

Bug fix release

  • Bug fix voor recipe id bestaat niet in db
  • Card weergaven op mobiel
  • Static file hosting via whitenoise ipv debug server

v0.0.1 beta

De allereeste versie van de hapmap Met functies als:

  • Overzicht van alle recepten
  • Ingredieten per recept per persoon
  • Basic back end voor makelijk toevoegen van recepten
